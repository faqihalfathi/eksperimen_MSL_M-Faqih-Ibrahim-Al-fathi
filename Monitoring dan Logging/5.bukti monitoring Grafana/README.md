## Cara mengambil screenshot:
1. Buka Grafana: http://localhost:3000 (default login: admin/admin)
2. Buat Dashboard baru dengan nama: [Username Dicoding Anda]
3. Tambahkan panel untuk setiap metrik
4. Data source: Prometheus (http://localhost:9090)
5. Screenshot setiap panel
